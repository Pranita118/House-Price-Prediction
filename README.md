
# 🏠 House Price Prediction — Internship Project Week 1

## 📌 Project Overview
A machine learning project that predicts house prices based on property 
features like area, number of rooms, location, and amenities using 
Linear Regression and Random Forest Regressor.

## 📦 Dataset
- **Source:** [Housing Prices Dataset — Kaggle](https://www.kaggle.com/datasets/yasserh/housing-prices-dataset)
- **File:** Housing.csv
- **Size:** 545 rows × 13 columns

## 🛠️ Libraries Used
- Pandas — Data loading and cleaning
- NumPy — Numerical operations
- Scikit-learn — ML models and evaluation
- Matplotlib / Seaborn — Visualizations

## ✅ Tasks Completed
- Task 1 — Data Loading & Exploration
- Task 2 — Data Cleaning & Encoding
- Task 3 — Model Building & Evaluation (Linear Regression + Random Forest)
- Task 4 — Visualizations (3 charts)
- Task 5 — Insights & Summary
